# Parent Teacher Conference Form

A clean and responsive Parent-Teacher Conference scheduling form built with HTML and CSS.

## Features
- Responsive design (works on mobile and desktop)
- Custom styled radio buttons using `::before` pseudo-element
- Smooth transitions and hover effects
- Clean, modern UI with a dark theme
- Accessible form structure with proper labels
